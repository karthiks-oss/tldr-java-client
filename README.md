# TLDR Java Client

This is a Java client for [tldr-pages](https://tldr.sh/)

## Usage

```
Usage: tldr [options] SEARCH

  Options:
    --help, -h
      Print help information and exit
    --platform, -p
      Platform [windows, macos|osx, linux]
    --version
      Print version and exit
```
